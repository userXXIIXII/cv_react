# Portfolio Aarin DEB

Bienvenue sur mon portfolio personnel. Ce projet est une application web moderne développée pour présenter mes réalisations, mes services et mon activité de développement.

## 🚀 Fonctionnalités
- **Interface responsive :** Adaptée aux mobiles et desktop grâce à Tailwind CSS.
- **Profil GitHub dynamique :** Récupération automatique de mes données via l'API GitHub avec gestion d'erreur (redirection 404).
- **Navigation fluide :** Utilisation de React Router pour une expérience sans rechargement.
- **Interaction utilisateur :** Footer avec bouton "retour en haut" natif.

## 🛠 Stack Technique
- React
- Tailwind CSS
- React Router
- GitHub API
